## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/AlvinGeorge-AG/ReadmeForge.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd ReadmeForge
    ```
3.  Install the dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

*   Run the application using `python main.py` or via the Dockerfile.

## Running Locally

*   Execute `python main.py` to start the application.

## Environment Variables

This repository does not explicitly show the use of environment variables.

## Deployment

*   Build the Docker image: `docker build -t readmeforge .`
*   Run the Docker container: `docker run -p 8000:8000 readmeforge`
